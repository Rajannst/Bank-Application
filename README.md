# Bank Application

## Description
A responsive banking web application built using Angular, featuring user authentication and basic banking operations.

## Features
- User Registration & Login
- Responsive UI
- Component-based architecture
- Clean and modular code

## Technologies Used
- Angular
- TypeScript
- HTML
- CSS
- Bootstrap

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `ng serve`
4. Open `http://localhost:4200`
